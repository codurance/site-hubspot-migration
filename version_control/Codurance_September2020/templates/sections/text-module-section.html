Discovery

 

Continuous Delivery was a major challenge for Roche. They operate in the highly regulated environment of healthcare. Roche started work to increase their overall agility and being able to frequently release safe changes while remaining fully compliant was a key part of this initiative. At Codurance, we routinely build state-of-the-art Continuous Integration and Deployment solutions, but the strong regulatory requirements presented a challenge that we relished. The proposed reference implementation would later be used as an example for similar applications. Thus, the solution also had to be very flexible, customisable and extendable, serving as the core template for any similar future migration in their portfolio.
